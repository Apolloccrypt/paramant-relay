Outlook add-in

ParaShare for Outlook

Replace email attachments with secure one-time download links. The receiver clicks, downloads, the file is destroyed. No storage, ever.

Compatibility
Outlook Web (OWA)
Outlook 365 Windows
Outlook 365 Mac
Outlook 2016 / 2019
Outlook mobile
Microsoft 365 Business
Requires a Microsoft 365 account (work, school, or personal). A PARAMANT API key is needed to send — receivers need no account.
Installation — 3 steps
1
Open Outlook add-in settings
Go to outlook.office.com → click the gear icon (top right) → View all Outlook settings → Mail → Customize actions → Manage add-ins.

Or go directly: outlook.office.com/mail/options/general/manage-apps →
2
Add from URL
Click + Add a custom add-in → Add from URL. Paste the manifest URL and click Install.
https://paramant.app/outlook/manifest.xml
If “Add from URL” is not available, use Add from file and upload the manifest below.
Download manifest.xml
3
Use it in compose
Open a new email in Outlook. Click Send via ParaShare in the toolbar ribbon. A panel opens on the right.

Enter your API key, select the attachments to replace, click Replace with ParaShare link →. Attachments are removed and secure download cards are inserted into your email body.

Deploy to your organisation

Roll out ParaShare to all users via the Microsoft 365 Admin Center — no individual installation needed.

1
Open Microsoft 365 Admin Center
Go to admin.microsoft.com → Settings → Integrated apps → Upload custom app
2
Upload the manifest
Paste the manifest URL or upload the file. Assign to users or groups.
https://paramant.app/outlook/manifest.xml
3
Done
ParaShare appears in Outlook for all assigned users within 24 hours. Contact privacy@paramant.app for enterprise rollout support.
