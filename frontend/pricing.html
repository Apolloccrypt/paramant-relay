Frequently asked questions

What is a relay? Do I need to run my own?
A relay is the server that temporarily holds encrypted blobs. You can use the hosted relay at relay.paramant.app (free, 5 keys), or self-host the relay on your own infrastructure using the Docker Compose stack and a plk_ license key.
What is a license key (plk_)?
A plk_ license key is an Ed25519-signed JWT you add to your self-hosted relay's .env file. It lifts the 5-key community limit and enables unlimited API keys. The relay verifies the signature on startup and on each hourly reload — no phone-home, no internet required.
Can I use the hosted relay for free?
Yes. The hosted relay at relay.paramant.app and sector relays (health, legal, finance, iot) are free to use. Free accounts are limited to 5 active API keys and standard rate limits. The relay operator never sees your plaintext — all encryption happens in your browser or SDK.
Is the source code available?
Yes. The relay, frontend, SDKs, and ParamantOS ISO are all open source on GitHub under BUSL-1.1. You can read every line of code that handles your data. Commercial use of the relay beyond the community limit requires a plk_ license.
What jurisdictions apply?
The hosted relay runs on servers in Germany (EU). It is subject to GDPR and German data protection law (BDSG). It is NOT subject to US CLOUD Act. Zero plaintext is stored — the relay only sees encrypted blobs which are deleted on first download or after TTL expiry.
Do you offer invoicing or bank transfer?
Yes. Email hello@paramant.app for an invoice. We support SEPA bank transfer for EU customers and bank wire for international orders.
