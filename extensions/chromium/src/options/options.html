Paramant Settings

Link expiry

How long a link stays downloadable. Each file is still single-download (burn-on-read); this is the upper limit before it is deleted. Your plan may cap the maximum.

Link format

How the link looks when inserted into your email.

Relay (advanced)

Leave blank to auto-detect the relay for your API key. Set this only if you run a self-hosted relay.

Transfer history

A local list of files you have sent (name, size, time). It never leaves this browser and never contains keys or links.
